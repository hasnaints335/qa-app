# Shopify QA Tool

Compare your Figma designs to live Shopify pages — get a visual diff instantly.

## How to use
1. Enter your Shopify page URL
2. Upload your Figma design export (PNG or JPG)
3. Click Run QA Comparison
4. Get a side-by-side view + difference heatmap + match score

## Deploy on Vercel
1. Upload this folder to GitHub
2. Go to vercel.com → Import project from GitHub
3. Click Deploy — done!
